Process

제작 과정은 짧고, 운영은 계속됩니다.

라이트 홈페이지는 빠르게 시안을 확인하고, 성장형·프리미엄은 필요한 기능 범위를 정리한 뒤 단계적으로 붙입니다.

STEP 01카카오톡 상담

업종, 목적, 필요한 기능, 희망 오픈일을 확인합니다.

STEP 02자료 전달

로고, 사진, 소개글, 메뉴, 연락처, 기존 사이트를 정리합니다.

STEP 031차 시안

라이트 기준 자료 준비 완료 후 최단 3일 안에 시안을 제공합니다.

STEP 04수정·피드백

기본 범위 안의 텍스트, 이미지, 섹션 조정은 무제한 반영합니다.

STEP 05오픈·검색 세팅

서버, SSL, 기본 SEO, Search Console, 문의 연결을 점검합니다.

STEP 06월간 운영

수정, 공지, 검색 상태, 기능 확장, AI 자동화까지 관리합니다.

FAQ

자주 묻는 질문

요금제 선택 전에 가장 많이 확인하는 조건을 정리했습니다.

정말 초기 제작비가 무료인가요?

네. 기본 요금제 범위 안에서는 초기 제작비 없이 월 구독료로 시작합니다. 기능 범위를 초과하는 개발은 별도 견적입니다.

월 5만원에 서버비도 포함되나요?

라이트 관리형에는 기본 서버비, SSL, 기본 유지관리가 포함됩니다. 트래픽 급증 또는 고사양 서버가 필요한 경우 별도 협의될 수 있습니다.

도메인 비용은 포함되나요?

도메인 구매와 갱신 비용은 별도입니다. 기존 도메인 연결 또는 신규 도메인 구매 안내를 지원합니다.

중간에 해지할 수 있나요?

해지는 가능합니다. 다만 제작·운영 구조상 최소 이용기간은 12개월을 권장합니다.

선제작 후결제가 가능한가요?

라이트 관리형 기본 범위는 상담 후 선제작 후결제 방식으로 진행할 수 있습니다. 완성 시안을 확인한 뒤 진행 여부를 결정할 수 있습니다.

최단 3일 제작이 가능한가요?

로고, 사진, 소개 문구 등 기본 자료가 준비된 라이트 홈페이지는 최단 3일 안에 1차 시안을 제공할 수 있습니다. 기능 개발이 포함되면 일정은 별도 산정합니다.

수정은 정말 무제한인가요?

기본 범위 안의 텍스트, 이미지, 섹션 순서, 간단한 문구 조정은 무제한 지원합니다. 신규 기능, 대량 페이지 추가, 외부 연동은 별도 범위입니다.

예약이나 결제 기능은 어떤 요금제부터 가능한가요?

예약, 신청폼, 결제, 알림톡, 관리자, 고객 데이터 관리는 성장 관리형부터 가능합니다.

AI 기능은 어떤 방식으로 들어가나요?

프리미엄 운영형에서 문의 분류, 답변 초안, 콘텐츠 초안, 견적 보조, 운영 리포트, 자동화 워크플로우 형태로 적용합니다.

기존 홈페이지를 이전할 수 있나요?

가능합니다. 기존 도메인, 콘텐츠, 이미지, 블로그, 검색 노출 상태를 확인한 뒤 이전 범위를 정합니다.
